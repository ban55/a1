About the Class...

Front-end frameworks focus on standards compliant responsive web development. Frameworks harness the power of HTML, CSS, and JavaScript to allow faster and more consistent design and development. This course will also examine the use of dynamic stylesheets such as LESS and SASS within the front-end framework.

Student Outcomes

Upon successful completion of this course, the student should be able to:

  1. explore a variety of front-end frameworks that are used to streamline the development of mobile-first websites;
  2. learn how the HTML, CSS and JavaScript components of front-end frameworks work;
  3. build flexible and complex responsive layouts using popular front-end frameworks;
  4. design layouts that adhere to a baseline grid and/or a fluid grid;
  5. build working prototypes with navigation, form, button elements, common CSS features, and UI components;
  6. create interactive content using the JavaScript plugins specific to the given framework;
  7. utilize the two most popular CSS preprocessors, Less and Sass;
  8. compare and contrast the strengths and weaknesses of a variety of front-end frameworks
